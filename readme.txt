== Description ==

It help your to view your recent popular post anywhere nicely. Where you want view your recent popular please paste this code here `<?php if(function_exists('round_top_posts')){round_top_posts();} ?> `.

== Installation ==

Step 1. Download the Plugin from  from official WordPress Plugin Directory.

Step 2. Upload it in you wp-content/plugins directory.

Step 3. Activate it from Admin panel.

Step 4. You can see "Round Top Post" option under Settings menu of your WordPress Admin panel. 

Step 5.  Where you want view your recent popular please past this code here
`<?php if(function_exists('round_top_posts')){round_top_posts();} ?> `.

Step 6. User with admin power can change the settings of the plugin.
